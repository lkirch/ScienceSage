## ğŸ·ï¸ Field Descriptions

- **feedback_id**: Unique identifier for the feedback entry (UUID).
- **timestamp**: When the feedback was submitted.
- **question**: The user's original question.
- **answer_level**: The requested answer complexity (e.g., "middle_school", "college", "advanced").
- **retrieved_chunks**: List of chunk IDs retrieved for this question.
- **llm_answer**: The answer generated by the LLM.
- **user_feedback**: User's rating (e.g., "upvote", "downvote").
- **comments**: Optional user comments about the answer.
